[1. Two Sum](https://leetcode.com/problems/two-sum/)

```python
nums = [3,3]
sum = 6
for i in range(len(nums)-1):
    for j in range(i+1,len(nums)):
        if sum == nums[i]+nums[j]:
            print(i,j)
```
