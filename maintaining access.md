**What it means:**  
The attacker tries to **stay inside the system for a long time** without being noticed.

**In simple terms:**  
Hiding a spare key inside the house so you can come back anytime.

**What this involves (high level):**

- Creating hidden access methods
    
- Avoiding detection
    
- Ensuring access even after reboots
    

**Why it matters:**  
Long-term access allows:

- Data theft
    
- Monitoring activity
    
- Repeated attacks
