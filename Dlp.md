Data Loss Prevention (DLP).

Data Loss Prevention (DLP) is a set of policies, processes, and tools designed to prevent sensitive data from being lost, leaked, or accessed by unauthorized usersâ€”whether accidentally or maliciously.

What DLP protects

<mark style="background: #FFB8EBA6;">Personal data</mark> (PII): names, SSNs, addresses

<mark style="background: #FFB8EBA6;">Financial data: </mark>credit cards, bank info

<mark style="background: #FFB8EBA6;">Health data: </mark>medical records (HIPAA)

<mark style="background: #FFB8EBA6;">Intellectual property</mark>: source code, designs, trade secrets


How DLP works (at a high level)

1. <mark style="background: #FFB8EBA6;">Identify & classify data</mark>
Find where sensitive data lives and label it (e.g., confidential, restricted).


2. <mark style="background: #FFB8EBA6;">Monitor data in motion, at rest, and in use</mark>

In motion: emails, uploads, network traffic

At rest: databases, file servers, cloud storage

In use: copying to USB, screenshots, printing



3. Enforce policies
Block, quarantine, encrypt, or alert when risky actions happen.


4. Alert & report
Notify security teams and create audit trails for compliance.
