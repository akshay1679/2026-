**What it is:**  
Tracking **all software** installed on your systems.

**Why it matters:**  
Malware often hides as “unknown software,” and outdated apps create vulnerabilities.

**In practice:**

- Allowlist approved software
    
- Block or remove unauthorized applications
    
- Track versions and patch levels
    

**Examples:**

- Prevent users from installing random apps
    
- Remove unsupported software like old Java versions
