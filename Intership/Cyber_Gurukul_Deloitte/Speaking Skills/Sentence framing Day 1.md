## Structure 1
self intro --> subject skill and action

<mark style="background: #FFB8EBA6;">i + skill/role + action/result</mark>

example: 
i am a python developer working on a machine learning algorithm.
i have basic knowledge of python im improving through projects

## Structure 2

while mentioning about the project

<mark style="background: #FFB8EBA6;">When/in my project --> i did --> outcome</mark>

example: In  my final year project  i worked on data compression algorithm which improved data retrieval and storage .

during my internship i worked an deep learning model which helps to predict the weather. 
## Structure 3

example: i believe teamwork is important because it improves efficiency

i have working knowledge of java and i used it in my academic projects
